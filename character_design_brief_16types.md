# LOVEQUESTプレイヤー図鑑 キャラクターデザインブリーフ（全16タイプ）

参考画像フォーマット準拠：メインイラスト（全身）＋ちびキャラ（チャーム/アイコン版）＋ロゴテキスト＋設定タグ

## 共通ルール（先に確認）

1. **カラーコード**：4元素で固定 — 🔥戦士＝レッド系(#E94560) / ☀️踊り子＝ゴールド系(#F5A623) / 🌿僧侶＝グリーン系(#4CAF73) / 🌙魔法使い＝ブルー・パープル系(#4A9EDB系)
   複合タイプは主属性のメインカラー＋副属性を差し色として1〜2箇所使用（世界観に応じてトーン調整可）
2. **構図**：全身メインイラスト＋左下にちびキャラ＋左上にロゴテキスト＋設定タグ
3. **ちびキャラ**：各タイプの「弱点」または「好物」を象徴する豆アイコンをセットで添える
4. **レア度**：`player_encyclopedia.html`と連動。ド直球＝★3、隣接軸の複合＝★4、対角（正反対軸）の複合＝★5
